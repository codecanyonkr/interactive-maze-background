Interactive Maze Background

The "Interactive Maze Background" is a customizable script that generates a new unique maze every time the page is loaded. The script can be easily placed into any existing web page to spruce up your website.

If you would like to travel through the maze yourself, just use the arrow keys to guide the avatar to the lower-right corner. You can click the peel in the upper-right corner to hide this content.

  • Works with any existing website
  • Fully customizable
  • Optional self-solver
  • Works in all major browsers with graceful degradation
  • Non-obtrusive since it waits for all content on the page to be loaded first before building the maze
  • Mazes are randomly generated for a new experience to each visitor
